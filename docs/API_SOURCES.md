## 자동 교차검증 로직 (index.js 내장)
- **주소 3중 대조**: 본점(금융위) vs 제조소(식약처) vs 사업장(연금) — 상충 시 자동 플래그
- **인원 2중 대조**: 연금 가입인원 vs 구인사이트 표기 (수동) — 비정규직 규모 추정
- **크로스체크 자동 등재**: gap / C등급 / 5년초과 / 주소상충 필드는 방문 확인 시트에 자동 포함
